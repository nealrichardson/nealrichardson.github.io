Using Travis-CI to Build Blogdown

Automate the building of your Rmarkdown static site with Travis-CI.
Published 19 May 2017

create repo on github git init git remote add origin git@github.com:nealrichardson/nealrichardson.github.io.git https://travis-ci.org/profile, enable

git checkout -b src (because it’s a user page, not repo page)

.gitignore .travis.yml DESCRIPTION generate key

gem install travis

travis encrypt –add (can’t do until you’ve enabled Travis on your repository and have git remote add origin)
